Overview

Guitar Lesson Difficulty Classifier ("the Extension") is committed to protecting your privacy. This extension does not collect, store, or share any personal information.

Data Collection

The Extension does NOT collect any of the following:

  • Personal information
  • Browsing history
  • Video watch history
  • User preferences or settings
  • Cookies or tracking data

How the Extension Works

When you visit a YouTube guitar lesson video, the Extension:

  • Reads the video title and description from the current page
  • Sends this text to our API for classification
  • Displays the difficulty result on the page
  • Does not store or log any of this information

Third-Party Services

The Extension uses a classification API hosted on Render.com to analyze video content. This API:

  • Receives video titles and descriptions for classification purposes only
  • Does not store or log any data
  • Does not track users
  • Does not share data with third parties

Permissions

The Extension requires the following permissions:

  • activeTab: To read video information and display badges on YouTube pages
  • scripting: To inject the badge display functionality into YouTube
  • host permissions (youtube.com): To run on YouTube video pages
